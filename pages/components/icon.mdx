#### Icon 图标

`svg`矢量图形，使用`Icon`图标组件，支持大小等属性自定义设置：

```js
import { Icon } from 'rJUI';
const a = () => {
    console.log('sss');
}
```

#### API

| 属性      | 描述      | 类型              | 默认  |
| --------- | --------  | ---------------  | ----  |
| className | 原生属性  | `string`          | -     |
| name      | 图标名称  | `number`          | -     |
| size      | 图标大小  | `number`、`string`| -     |
| rotate    | 旋转角度  | `number`          | 0     |
| spin      | 旋转动画  | `boolean`         | false |
| style     | 图标样式  | `CSSProperties`   | -     |
| width     | 图标宽度  | `number`、`string`| -     |
| height    | 图标高度  | `number`、`string`| -     |
| color     | 图标颜色  | `string`          | -     |
